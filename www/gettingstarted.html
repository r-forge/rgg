Getting started!

RGG requires Java version 5 or later. It can be used in two different ways:

1. Standalone

The standalone version of RGG  can be used to open an .rgg file and to execute the generated R script directly from the RGG. To download RGG standalone, please go to the "Download" section. After download follow these instructions to run RGG.

  • On Windows and Linux: Download and exract the rgg.zip, and then go to the /bin directory. Use one of the executables rgg.exe for Windows and rgg (a shell script) for Linux.
  • Mac OS X: Download and extract the rgg.app.zip, and then click twice on rgg.app
Requirements to run RGG standalone:
  1. Java Runtime Edition (JRE) 5 or later (www.java.com)
  2. R (http://www.r-project.org)
RGG standalone uses the "Rscript" command, found in the /bin directory of R installation folder,   to execute the generated R script. If this command is not found in the PATH environment variable, user will be prompted to enter the installation folder (e.g. on Windows "C:\Program Files\R\R-2.6.1\bin").

Here is a screenshot of the standalone version of RGG. An .rgg file (qcreport.rgg) is already loaded and executed here.

Figure 1: RGG standalone

The main window consists of two parts, left (GUI) and right (output), which can be rearranged by drag and drop. An .rgg file is loaded, either by clicking the "Open RGG" button or using the shortcut "CTRL-O". After setting inputs, parameters etc. on the GUI, the generated R script can be executed by clicking the "Run" button or using the "CTRL-R" shortcut. The output from R is displayed on the "Output" window. 

2. As an R package within JGR

The second way to use RGG is by installing it as an R Package in JGR. Here JGR provides the connection to R. JGR is a Java GUI for R. For installation of JGR, please visit the JGR homepage.

After you have successfully installed JGR, please execute the following R code inside JGR. It will install RGG for you. 

install.packages("rgg",repos="http://r-forge.r-project.org")

Then load the RGG package by calling "library(RGG)". This will add a menu "RGG" on the menubar of JGR window (see the screenshot below).

To open an RGG file click on the submenu "Load" of "RGG". Let's load the same RGG file, which we have opened in the stand-alone version.

The user interface is the same except for the two buttons at the bottom: "Run" and "Open in Editor". The "Run" button generates the corresponding R code and sends it to R for execution. "Open in Editor" will open the generated R code in the JGR editor, which uses R specific highlighting.
