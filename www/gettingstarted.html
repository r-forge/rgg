Getting started!

RGG can be used in two different ways:

1. RGG as an R package within JGR

The first way to use RGG is by installing it as an R Package in JGR. Here JGR provides the connection to R. JGR is a Java GUI for R.

Installation

1. Download and install R (2.6.x or newer)

2. Install Java (5 or newer)

3. Download JGR 1.5 or higher.

4. Install RGG by executing the following lines within JGR or download it from R-Forge:

 install.packages("RGG",repos="http://r-forge.r-project.org")

Notice that this will work only, if the latest version of R is installed, at this moment 2.6.x. If not, please go to the Downloads section and download an appropriate RGG package archive and install it manually.

Then load the RGG package by calling "library(RGG)". This will add a menu "RGG" on the menubar of JGR window (see the screenshot below).

To open an RGG file click on the submenu "Load" of "RGG". Let's load an RGG file, qcreport.rgg.

The user interface has two buttons at the bottom: "Run" and "Open in Editor". The "Run" button generates the corresponding R code and sends it to R for execution. "Open in Editor" will open the generated R code in the JGR editor, which uses R specific highlighting.

2. Standalone 

Installation

1. Download and install R (2.6.x or newer)

2. Install Java (5 or newer)

3. Install RGG:
3.1 Web Start: Click here to install automatically.
3.2 Manual installation:
Windows and Linux: Download the rgg.zip. After extraction, please go to /bin directory and use the appropriate launcher to run RGG.
Mac OS X: Coming soon... For now, please use "Web Start".

The standalone version of RGG  can be used to open an .rgg file and to execute the generated R script directly from the RGG.
RGG standalone uses the "Rscript" command, found in the /bin directory of R installation folder,   to execute the generated R script. If this command is not found in the PATH environment variable, user will be prompted to enter the installation folder (e.g. on Windows "C:\Program Files\R\R-2.6.2\bin").

Here is a screenshot of the standalone version of RGG. An .rgg file (qcreport.rgg) is already loaded and executed here.

Figure 1: RGG standalone

The main window consists of two parts, left (GUI) and right (output), which can be rearranged by drag and drop. An .rgg file is loaded, either by clicking the "Open RGG" button or using the shortcut "CTRL-O". After setting inputs, parameters etc. on the GUI, the generated R script can be executed by clicking the "Run" button or using the "CTRL-R" shortcut. The output from R is displayed on the "Output" window.
