Downloads

RGG Standalone

Windows and Linux: Download the rgg.zip. After extraction, please go to /bin directory and use the appropriate launcher to run RGG.

Mac OS X: Download the rgg.app.zip. After extraction, click twice on rgg.app.

RGG R package for JGR

RGG can be used together with JGR. To install the R package over internet, please execute the following lines:

install.packages("rgg",repos="http://r-forge.r-project.org")

Or download the zip archive from R-Forge
