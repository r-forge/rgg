Downloads

RGG as R package for JGR

1. Download and install R (2.7.0 or higher is recommended)

2. Install Java (5 or newer)

3. Download JGR 1.5 or higher.

4. Install RGG
4.1 By executing the following lines within JGR 

 install.packages("RGG",repos="http://R-forge.R-project.org")
4.2 Or download the package and install it manually:
Package source (.tar.gz) | Windows binary (.zip) | MacOS X binary (.tgz)

RGGRunner

1. Download and install R (2.7.0 or higher is recommended)

2. Install Java (5 or newer)

3. Install RGG:
3.1 Web Start: Click here to install automatically. It will put a shortcut on the desktop. Next time it can be run directly using it. Will be available soon!
3.2 Manual installation:
Windows and Linux: Download the rggrunner.zip. After extraction, please go to /bin directory and use the appropriate launcher to run RGG.
MacOS X: Download the rggrunner.dmg and double click to mount the image. After mounting, please double click again on the "RGG" icon to run the application.
